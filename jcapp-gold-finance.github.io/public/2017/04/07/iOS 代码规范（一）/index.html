1.在类的头文件中尽量少引入其他头文件

要点

  • 除非确有必要，否则不要引入头文件。一般来说，应在某个类的头文件中使用向前声明来提及别的类，并在实现文件中引用那些类的头文件==。（@class）
  • 协议尽量移动至“class-continuation 分类”（==.m 文件中的intreface==）
